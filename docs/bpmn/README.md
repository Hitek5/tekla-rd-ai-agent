# BPMN 2.0 process descriptions

These files are BPMN 2.0 XML process descriptions that can be opened in bpmn.io, Camunda Modeler, Bizagi Modeler or similar tools.

- `chat-tool-approval.bpmn` - user request, RAG, LLM, policy, approval and Tekla tool execution.
- `rag-ingestion.bpmn` - document intake, security review, chunking, metadata and retrieval publication.
- `airgap-rollout.bpmn` - internet-side artifact preparation and closed-contour import.

The BPMN files are deliberately conservative: they describe the approval gates and handoffs that must remain visible during the MVP and pilot.
